# 简介

> z-wechat-bot 是基于原项目修改的大模型智能对话机器人，支持微信公众号、企业微信应用、飞书、钉钉接入，可选择GPT3.5/GPT4.0/Claude/Gemini/ChatGLM/KIMI/文心一言/讯飞星火/通义千问，能处理文本、语音和图片，通过插件访问操作系统和互联网等外部资源，支持基于自有知识库定制企业AI应用。
